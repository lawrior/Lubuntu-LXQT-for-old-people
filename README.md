# how to copy if acces denied
1. method: Open 'File Manager' (PCManFM) go to 'Tool' tab and choose 'Open as Administrator'.
2. method: Open 'Terminal' example: cp -r /home/USER/Downloads/Win10 /usr/share/lxqt/themes/
